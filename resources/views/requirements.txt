1️⃣ Install Tesseract OCR

Download the Windows installer from the official repo:
https://github.com/tesseract-ocr/tesseract

Or directly from: UB Mannheim builds

Run the installer and note the installation path, e.g.:

C:\Program Files\Tesseract-OCR

2️⃣ Add Tesseract to your PATH

Press Win + S → type Environment Variables → Open Edit the system environment variables → click Environment Variables.

Under System variables, find Path → Edit → Add a new entry:

C:\Program Files\Tesseract-OCR


Click OK and restart your terminal or VS Code to apply.

3️⃣ Verify installation

Open Command Prompt and run:

tesseract --version


You should see something like:

tesseract 5.3.0
